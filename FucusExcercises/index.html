منصة تمارين التركيز والانتباه

صفحة أساسية موحدة لكل التمارين، مع ثيمين (ليلي/نهاري) ومنطقة عرض واحدة تستقبل أي تمرين سكربت لاحقًا.

التدريبات

كل تمرين سيستخدم نفس هذا القالب: اسم، تعريف، فائدة، ومدة.

بعد بناء التمارين، سيظهر هنا وصف مختصر يوضح طبيعة التمرين بلغة بسيطة.

تعريف التمرين

هذه منطقة تعريف عامة. لكل تمرين لاحقًا نص تعريفي قصير يوضح ما يراه أو يفعله المتدرّب أثناء الجلسة.

الفائدة الإدراكية

هنا يتم توضيح الفائدة بشكل موجز (مثلاً: تعزيز الانتباه البصري، تقوية الذاكرة اللحظية، زيادة القدرة على ملاحظة التفاصيل الدقيقة...).

مدة الجلسة المقترحة

لاحقًا يمكن تحديد مدد مختلفة لكل تمرين (٢–٣–٥ دقائق مثلًا)، مع إمكانية اختيار المدة قبل البدء.

منطقة عرض التمرين أي سكربت تمرين لاحقًا سيحقن محتواه هنا فقط.
